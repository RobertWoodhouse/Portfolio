DATER

JAVA | SWING

Dater is a room renting desktop application created in Java using Swing GUI.

The user selects from a list of criteria picking the categories that best match the type of roommate they are willing to rent a room to e.g. smoker, pets, and gender.

From the selected criteria a list of roommates is produced starting with the most compatible match, the user then has the option to contact a potential roommate from this list.
