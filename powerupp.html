POWER UPP

C# | .NET | XAML | EXCEL

Power Upp is a fitness tracker app focusing on calisthenics workout plans. The purpose of this app is to track your weekly workout routine and generate an Excel line graph from the gathered data.

The user selects an exercise type from the dropdown menu e.g. Squats, Push ups. The user then selects the number of sets they performed for their exercise, finally they enter the number of reps they did for each set.

Once all options have been selected a line graph is generated showing the users progress for each exercise performed and the date the exercise was recorded.
